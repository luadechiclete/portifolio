20 comandos mais essenciais em javascript

04 de janeiro de 2023

O JavaScript é uma linguagem de programação fundamental para qualquer desenvolvedor web. Existem muitos comandos e recursos que podem ser úteis na hora de criar aplicações e sites, mas alguns são mais essenciais do que outros. Neste artigo, vamos falar sobre os 20 comandos mais importantes em JavaScript que todo desenvolvedor deve conhecer.

  • console.log() - permite imprimir mensagens no console do navegador, o que é útil para depurar código e verificar o valor de variáveis.
  • document.querySelector() - permite selecionar um elemento HTML a partir de seu seletor CSS.
  • element.addEventListener() - adiciona um event listener a um elemento HTML, permitindo que o código JavaScript seja executado quando o evento ocorrer.
  • for loop - permite iterar sobre uma sequência de valores, como um array ou um objeto.
  • if statement - permite executar um código específico se uma determinada condição for verdadeira.
  • function - permite criar uma função que pode ser chamada repetidamente para executar uma tarefa específica.
  • let - permite declarar uma variável localmente em um bloco de código.
  • const - permite declarar uma constante, ou seja, uma variável cujo valor não pode ser alterado.
  • array.forEach() - permite iterar sobre os elementos de um array e executar uma função para cada um deles.
  • array.map() - permite iterar sobre os elementos de um array, transformá-los e retornar um novo array com os elementos transformados.
  • array.filter() - permite iterar sobre os elementos de um array e retornar um novo array com os elementos que atendem a uma determinada condição.
  • array.reduce() - permite iterar sobre os elementos de um array e retornar um único valor resultante, a partir da aplicação de uma função específica.
  • object.keys() - retorna um array com as chaves de um objeto.
  • object.values() - retorna um array com os valores de um objeto.
  • object.entries() - retorna um array de arrays, onde cada subarray contém uma chave e um valor de um objeto.
  • JSON.parse() - permite converter uma string em formato JSON em um objeto JavaScript.
  • JSON.stringify() - permite converter um objeto JavaScript em uma string no formato JSON.
  • parseInt() - permite converter uma string em um número inteiro.
  • parseFloat() - permite converter uma string em um número de ponto flutuante.
  • isNaN() - permite verificar se um valor não é um número.


    • Esses são apenas alguns dos comandos mais importantes em JavaScript, mas existem muitos outros que também são muito úteis.
